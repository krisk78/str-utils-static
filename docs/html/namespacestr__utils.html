std::wstring str_utils::str_to_wstr ( const std::string &  str,
unsigned int  codepage 
)

Converts a string to utf-16 wide string using the specified codepage.

Warning
This function is only implemented for Windows platforms.
Parameters
strThe string to convert
codepageThe codepage to use
Returns
an utf-16 wide string equivalent to str
See also
This function uses the function MultiByteToWideChar. Refer to MultiByteToWideChar function (stringapiset.h) (Microsoft) for details on codepage identifiers.

Note that the result string must be converted to a c-string with std::string::c_str() to get a LPCWSTR string.
